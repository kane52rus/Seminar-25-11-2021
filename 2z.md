### Создание коммитов
Для создания коммитов используется команда *git commit*. Для этого в терминале с папокой-репозиторием необходимо написать команду *git commit -m <сообщение к коммиту>*. Сообщение к комиту писать ***ОБЯЗАТЕЛЬНО***
